## dotfiles

vim version>=8.1 

- vimrc

  vim config

- ideavimrc

  IDEA vim plugin config

- settings.json && keybindings.json

  vscode vim plugin config


### usage

`make init`

`:PlugInstall` install plugins

- coc.vim

vim >= 8.1 && nodejs >= 14.14:

```
:CocInstall coc-json coc-tsserver
```
